# Active Directory / Access Management Simulation

## Overview
This project simulates real-world identity and access management tasks commonly handled by IT Help Desk and junior systems support teams in Active Directory-based environments.

## Objective
To demonstrate practical understanding of user account provisioning, group-based access control, password resets, account lockouts, onboarding/offboarding workflows, and MFA support through structured documentation and realistic support artifacts.

## Included Scenarios
- New user account requests
- Department-based group assignments
- Password reset and account unlock support
- MFA enrollment assistance
- User onboarding workflow
- User offboarding workflow
- Access control matrix and least-privilege review
- Account lockout resolution documentation

## Files Included
- `access-requests.xlsx` — simulated user/account access requests
- `password-reset-log.xlsx` — password reset and account unlock cases
- `onboarding-checklist.xlsx` — new user provisioning workflow
- `offboarding-checklist.xlsx` — account deprovisioning workflow
- `access-matrix.xlsx` — department-to-access mapping
- `lockout-resolution-report.docx` — formal account lockout case documentation
- `mfa-enrollment-guide.docx` — MFA support guidance
- `screenshots/` — screenshots of project artifacts

## Skills Demonstrated
- Active Directory support concepts
- Identity and access management (IAM)
- Group-based access control
- Password reset procedures
- Account lockout handling
- User onboarding and offboarding
- Least-privilege access planning
- IT documentation and ticket workflow

## Notes
This is a structured simulation project designed to reflect common front-line IT support responsibilities in Windows/Active Directory environments without requiring a dedicated virtual lab.

## Resume Value
This project demonstrates the support-side operational knowledge of Active Directory environments that is highly relevant for entry-level IT Help Desk, desktop support, and junior IT support roles.
